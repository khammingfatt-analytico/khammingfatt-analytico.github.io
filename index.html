    Project 2: Computer Vision Project
    AI Squat Posture Assistant with MediaPipe and OpenCV

    The AI Squat Training Assistant project leverages machine learning to improve individuals' squat techniques, offering a tool for performing correct squats independently, assessing performance, and providing feedback to enhance form. Utilizing OpenCV for real-time body pose data collection via webcam and MediaPipe for landmark detection, the project gathered coordinates for 33 body landmarks, saving them in a dataset for model training. Five classification models, including logistic regression, random forest, XGBoost, Adaboost, and a Voting Classifier, were employed for high accuracy in posture classification. This innovative solution empowers users to learn correct squatting independently, assess their performance, and mitigate injury risks, contributing to better fitness and well-being.

    Key Skills Demonstrated:

    • Real-time Data Collection with OpenCV
    • Landmark Detection using MediaPipe
    • Machine Learning Ensemble Models with Logistic Regression, Random Forest, XGBoost and AdaBoost
    • Model Evaluation and Validation with metrics like Accuracy and F1 score
    • Developing algorithms to provide prescriptive feedback based on model predictions to correct squat posture.
    • Deployment with Streamlit

    Project 3: Regression Project with Interactive Map
    Data Backed Solutions for Combating WNV in Chicago

    The project focused on combating West Nile Virus (WNV) in Chicago through data science. Utilizing datasets from mosquito traps, weather conditions, and spray efforts, we developed machine learning models to predict WNV presence. Key techniques included One Hot Encoding for categorical data, SMOTE for class balance, and Standard Scaling for feature normalization. Models like logistic regression, random forest, XGBoost, Adaboost, and a Voting Classifier were evaluated, showing significant predictive capability. The project's insights aim to guide CDC's WNV control strategies, emphasizing cost-effective measures, targeted efforts during peak seasons, and focusing on areas with high Culex pipiens populations to reduce WNV spread efficiently.

    Key Skills Demonstrated:

    • Data Preprocessing with techniques like One Hot Encoding, Synthetic Minority Oversampling Technique (SMOTE) and Standard Scaling
    • Machine Learning Modeling with logistic regression, random forest, XGBoost, and Adaboost
    • Model Evaluation with True Positive Rate (TPR), True Negative Rate (TNR) and Receiver Operating Characteristic (ROC) curves
    • Potential Cost Cutting using Geospatial Analysis and Cost-Benefit Analysis
    • Interactive Data Visualization with the Streamlit Deployment
